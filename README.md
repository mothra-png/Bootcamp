# Bootcamp

🎬 Movie Watchlist Smart Contract
📌 Description

This project is a simple decentralized movie watchlist application built using Soroban smart contract on Stellar.
The purpose of this app is to help users store a list of movies they want to watch directly on the blockchain.

Each movie has a unique ID, a title, and a status that shows whether the movie has been watched or not.

⚙️ How It Works

The smart contract stores all movie data in blockchain storage using a vector (list).
Every time a user adds, updates, or deletes a movie, the contract will update the stored data.

A random ID is generated for each movie to make sure every item is unique.

✨ Features
Add a new movie to the watchlist
View all saved movies
Mark a movie as watched
Delete a movie from the list
